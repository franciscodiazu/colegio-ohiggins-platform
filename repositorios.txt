Repositorio principal: https://github.com/franciscodiazu/colegio-ohiggins-platform
Descripción: Monorepo del proyecto "Colegio O'Higgins Platform" que contiene frontend, BFF, microservicios, paquetes y archivos de infraestructura.

Componentes (subdirectorios en el repositorio principal):

- Frontend: https://github.com/franciscodiazu/colegio-ohiggins-platform/tree/main/frontend
  Descripción: Aplicación cliente en React + Vite; incluye `package.json`, `src/`, `public/` y README con instrucciones de instalación/ejecución.

- Admin Server: https://github.com/franciscodiazu/colegio-ohiggins-platform/tree/main/admin-server
  Descripción: Spring Boot Admin Server para monitoreo de microservicios; contiene `pom.xml`, `src/` y README.

- Backend BFF: https://github.com/franciscodiazu/colegio-ohiggins-platform/tree/main/backend-bff
  Descripción: BFF en Spring Boot (Maven). Contiene `pom.xml`, `src/`, `mvnw` y README con comandos de ejecución.

- Microservicio Asistencia: https://github.com/franciscodiazu/colegio-ohiggins-platform/tree/main/ms-attendance
  Descripción: Microservicio Spring Boot para gestión de asistencia; incluye `pom.xml`, `src/`, `run-tests.ps1` y README.

- Microservicio Estudiantes: https://github.com/franciscodiazu/colegio-ohiggins-platform/tree/main/ms-students
  Descripción: Microservicio Spring Boot para gestión de estudiantes; incluye `pom.xml`, `src/`, `run-tests.ps1` y README.

- Infra / Docker: https://github.com/franciscodiazu/colegio-ohiggins-platform/tree/main/Infra
  Descripción: Archivos de infraestructura y Docker (docker-compose, Dockerfiles, init.sql para MySQL).
