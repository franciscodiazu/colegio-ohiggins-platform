# @colegio-ohiggins/ui

Componentes UI reutilizables para el frontend del proyecto.

## Uso

```jsx
import { SchoolHeader, TopNavTabs } from '@colegio-ohiggins/ui';
import '@colegio-ohiggins/ui/styles.css';
```

## Build de libreria

```bash
npm run build
```

Esto genera los artefactos en `dist/` para publicar en NPM.
